# 🏅 Olympics Data Analysis: 120 Years of History

An interactive, data-driven web application designed to analyze and visualize Olympic Games data from Athens (1896) to Rio (2016). Built by **Asmit Prem Kumar**.

---

## 🚀 Live Demo
Experience the interactive dashboard here:  
https://olympics-data-analysis-1896-2016.streamlit.app/

---

## 📌 Project Overview
The "Olympics Analysis" app transforms raw historical data into actionable insights. It addresses the complexity of tracking 120+ years of athlete performance, medal tallies, and country-wise growth through a seamless user interface.



### Key Modules:
* **Medal Tally:** A dynamic leaderboard filtered by Year and Country.
* **Overall Analysis:** High-level statistics on participating nations, events, and athletes over time.
* **Country Profiling:** Longitudinal study of a specific nation's performance including "Medal Heatmaps."
* **Athlete Insights:** Analysis of physical attributes (Age/Height/Weight) vs. medal success.

---

## 💻 Tech Stack & Tools
* **Frontend:** `Streamlit` (Framework for Rapid Data Apps)
* **Data Processing:** `Pandas`, `NumPy`
* **Visualization:** `Plotly` (Interactive), `Seaborn`, `Matplotlib`
* **Deployment:** `Streamlit Cloud` 

---

## 📊 Technical Implementation
The application follows a modular functional approach:
1.  **Data Preprocessing:** Cleaning the `athlete_events.csv` and `noc_regions.csv` to handle missing values and duplicate entries.
2.  **Helper Functions:** Specialized logic for fetching medal tallies and athlete-wise statistics.
3.  **Visualizations:** Use of Plotly's `line_chart` and `distplots` to ensure 100% interactivity for the end-user.

---

## 🛠️ How to Use
1. **Navigate** through the sidebar to select different analysis modes.
2. **Filter** by year, sport, or country to get specific insights.
3. **Interact** with the Plotly graphs to see detailed statistics on hover.
